Linux main apt repo for AI-Speaker / Asystent domowy

Stable APT repository

This apt repository contains packages for current stable release under the /dists/dom/stable directory.

# sources.list for the main stable AI-Speaker repository:
deb [trusted=yes] https://powiedz.co/apt dom stable

Beta APT repository

This apt repository contains packages for current test release under the /dists/dom-dev/beta directory.

# sources.list for the main beta AI-Speaker repository:
deb [trusted=yes] https://powiedz.co/apt dom-dev beta
